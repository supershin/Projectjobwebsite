ถ้าคุณเห็นหน้านี้สวยงาม แสดงว่า:

  • Bootstrap 5 โหลดสำเร็จ
  • Bootstrap Icons โหลดสำเร็จ
  • CSS Inline ทำงานได้

ปัญหาคือ: ไฟล์ css/style.css อาจไม่โหลด เพราะคุณเปิดไฟล์แบบ file:// ให้ลองวิธีแก้ด้านล่าง

🔧 วิธีแก้ไข:
  1. ใช้ VS Code: ติดตั้ง Extension "Live Server" แล้วคลิกขวาที่ index.html เลือก "Open with Live Server"
  2. ใช้ Python: เปิด Terminal ที่โฟลเดอร์ public แล้วรัน: python -m http.server 8000 แล้วเปิด http://localhost:8000
  3. ใช้ Node.js: ติดตั้ง npm install -g http-server แล้วรัน http-server ที่โฟลเดอร์ public
