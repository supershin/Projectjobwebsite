วิธีเปิด Preview

เนื่องจากเว็บไซต์ใช้ external CSS และ JavaScript ต้องเปิดผ่าน Web Server เท่านั้น (ไม่สามารถเปิดแบบ file:// ได้)

วิธีเปิดดู (เลือก 1 วิธี)
1
VS Code + Live Server (แนะนำ)
  1. เปิด VS Code
  2. ติดตั้ง Extension: Live Server by Ritwick Dey
  3. คลิกขวาที่ index.html
  4. เลือก "Open with Live Server"
2
Visual Studio 2022
  1. เปิด Visual Studio 2022
  2. File → Open → Folder → เลือก folder public
  3. คลิกที่ปุ่ม "Open in Browser" หรือกด Ctrl + Shift + W
3
Python Simple HTTP Server

เปิด Terminal/CMD ที่ folder public แล้วรัน:

python -m http.server 8000

จากนั้นเปิดเบราว์เซอร์: http://localhost:8000

4
Node.js http-server

เปิด Terminal/CMD ที่ folder public แล้วรัน:

npx http-server -p 8000

จากนั้นเปิดเบราว์เซอร์: http://localhost:8000
